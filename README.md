# AI Sentiment & Emotion Analysis

#Overview -

This project is a web-based AI application that analyzes user text and predicts sentiment and emotion in real-time.
It uses pretrained transformer models to understand context and provide accurate results along with confidence scores and visualizations.


 #Features
 
o AI-based Sentiment Analysis (Positive, Negative, Neutral)

o Emotion Detection (Happy, Sad, Angry, Fear, Empathy)

o Interactive Charts using Chart.js

o Confidence Score Display

o Context-aware predictions (handles phrases like “not good”)

o Input History Tracking

o Real-time Web Interface



 #Tech Stack

PyTorch

🔹 Frontend

HTML

🔹 Visualization

Chart.js



#Workflow

1 User enters text

2 Frontend sends request using Fetch API

3 Flask backend receives request

4 AI model processes text

5 Sentiment & emotion are predicted

6 Results returned with confidence score

7 Frontend displays results and charts
